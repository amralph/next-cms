## Overview

A content management system (CMS) built with Next.js and Postgresql. This project allows users to manage content dynamically and fetch it with server-side rendering or API routes.

## Demo

[Demo](https://youtu.be/3kHgjcW3iX4)

## Features

- Separate and private workspaces for different users
- Various field types including files, rich text, references to other documents, arrays, and other basic types
- RESTful API with filters and field selection
